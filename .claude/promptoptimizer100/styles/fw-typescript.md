# fw-typescript

Apply TypeScript patterns:
- Enable strict mode: strictNullChecks, noImplicitAny, strictFunctionTypes
- Use generic constraints with extends for type-safe abstractions
- Model state with discriminated unions and exhaustiveness checking
- Write type guards (is, asserts) for runtime type narrowing
- Use utility types (Partial, Pick, Omit, Record) to transform existing types
- Create declaration files (.d.ts) for ambient type declarations
- Use branded types for nominal typing of primitive-based values
