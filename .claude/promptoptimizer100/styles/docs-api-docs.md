- List all endpoints grouped by resource with HTTP methods and paths
- Include request/response examples for every endpoint with status codes
- Document authentication requirements per endpoint (public, authenticated, admin)
- List all error codes with their meaning and recovery suggestions

- Provide SDK examples in at least two languages (curl + one programming language)
- Include parameter tables with types, constraints, defaults, and nullability
