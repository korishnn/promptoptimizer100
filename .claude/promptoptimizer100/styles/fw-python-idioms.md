# fw-python-idioms

Write Pythonic code:
- Use list comprehensions and generator expressions over map/filter with lambdas
- Context managers (with statement) for resource cleanup and transactions
- Decorators for cross-cutting concerns: logging, timing, caching, auth
- Dataclasses for simple data containers with less boilerplate
- Type hints everywhere for better tooling and documentation
- Prefer EAFP (Easier to Ask Forgiveness than Permission) over LBYL
- Leverage itertools and collections for common iteration and data structure patterns
