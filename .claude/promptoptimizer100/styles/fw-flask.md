# fw-flask

Apply Flask best practices:
- Use the application factory pattern for configurable app instances
- Organize with Blueprints for modular route grouping by domain
- Integrate SQLAlchemy with Flask-SQLAlchemy for database access
- Validate requests with Marshmallow schemas or Pydantic models
- Implement global error handlers for consistent JSON error responses
- Create custom CLI commands with Click for management tasks
- Use separate testing configuration with an in-memory SQLite database
